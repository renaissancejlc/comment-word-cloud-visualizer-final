# Youtube Comment Cloud

Youtube Comment Cloud is a minimalist, brutalist-inspired, browser-based video comments visualizer app. It supports Youtube video urls, analyzes top occuring words in comments of that video, and displays an arrangement of those words where size represents the occurence frequency.

https://youtubecomment.cloud/

## Getting Started

1. Install dependencies:

```
npm install
```

2. Run the development server:

```
npm run dev
```

3. Open in browser at:

```
http://localhost:5173
```
